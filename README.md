# json2every
一行json，生成一个页面或者系统

## RFC
[rfc-001](./rfcs/rfc-001.md)
## Quick Start

```bash
# Install Dependencies.
$ pnpm i 
# Build and link packages.
$ pnpm packages:build
# Start watch service
$ pnpm start
```
